# Frontend

## Guia do desenvolvedor

Préquisitos
* [npm](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)
* [yarn](https://classic.yarnpkg.com/lang/en/docs/install/)

Executando o linter

```shell
yarn lint
```

Formatando o código

```shell
yarn format
```

Iniciando o servidor

```shell
yarn start
```
